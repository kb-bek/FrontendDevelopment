Графический дизайн

Графический дизайн — художественно-проектная деятельность по созданию гармоничной и эффективной визуально-коммуникационной среды. С другой стороны, графический дизайн − это форма визуальной коммуникации с использованием текста, изображений или продвижения посыла для представления информации.
